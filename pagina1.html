componentes de una computadora

PSU | Placa base | NIC |

¿Qué es una fuente de poder (PSU)?

La fuente de poder (PSU) convierte la electricidad de tu casa en energía utilizable para los componentes de tu computadora, y protege contra fluctuaciones eléctricas. Es esencial para el funcionamiento seguro y estable del sistema.
Es un dispositivo que transforma la corriente alterna (AC) de la red eléctrica en corriente continua (DC), que es la forma de energía que usan los componentes internos de la computadora.
Se encarga de distribuir voltajes específicos (como 3.3V, 5V y 12V) a diferentes partes del sistema, como la placa base, el procesador, la tarjeta gráfica y los discos duros.
Alimentar todos los componentes del sistema con el voltaje adecuado.
Proteger contra sobrecargas, cortocircuitos y picos de tensión, evitando daños a los componentes.
Regular la energía para mantener un flujo estable, lo cual es vital para evitar reinicios inesperados o fallos de hardware.

¿Qué es la placa base?

Es una tarjeta de circuito impreso que sirve como plataforma para conectar el procesador, la memoria RAM, la tarjeta gráfica, el almacenamiento y otros periféricos.
También se le conoce como placa madre, placa principal o mainboard.
Interconexión: Permite que todos los componentes se comuniquen entre sí.
Distribución de energía: Recibe energía de la fuente de poder y la distribuye a los componentes conectados.
Gestión de datos: Controla el flujo de datos entre el procesador, la memoria y los dispositivos de entrada/salida.
Configuración del sistema: Incluye el BIOS/UEFI, que permite configurar parámetros básicos del hardware.
Socket del procesador: Donde se instala la CPU. Su tipo determina qué procesadores son compatibles.
Ranuras de RAM: Para instalar módulos de memoria.
Chipset: Controla la comunicación entre el procesador y otros componentes. Define capacidades como número de puertos USB, soporte para discos NVMe, etc.
Ranuras PCIe: Para tarjetas gráficas, de sonido, de red, etc.
Conectores SATA/M.2: Para discos duros y unidades SSD.
Puertos traseros: USB, HDMI, Ethernet, audio, etc., para conectar periféricos externos.

¿Qué es una NIC?

Una Network Interface Card (NIC), también conocida como adaptador de red o controlador de red, es una tarjeta o chip que permite que un dispositivo se comunique con una red. Puede estar integrada en la placa base o ser una tarjeta independiente conectada a una ranura PCIe.
Conexión a redes: Permite que la computadora se conecte a redes LAN, WAN o internet.
Transmisión y recepción de datos: Convierte los datos del sistema en señales eléctricas (o inalámbricas) y viceversa.
Identificación única: Cada NIC tiene una dirección MAC única que identifica al dispositivo en la red.
Gestión de protocolos: Trabaja en las capas física y de enlace del modelo OSI, gestionando la entrega de paquetes de datos.
NIC cableada (Ethernet):
Usa cables de red (como RJ-45).
Ofrece mayor estabilidad y velocidad.
Común en PCs de escritorio y servidores.
NIC inalámbrica (Wi-Fi):
Usa señales de radio para conectarse a redes Wi-Fi.
Ideal para laptops y dispositivos móviles.
Puede ser interna o externa (USB).
NIC de fibra óptica:
Utiliza cables de fibra para conexiones de alta velocidad y larga distancia.
Común en centros de datos y redes empresariales.
